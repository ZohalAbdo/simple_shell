# SIMPLE SHELL
A shell is a program that executes other programs in response to text commands.

## Description

## Run
./hsh

## Test Command
1. For list directories or file:
* ls -l
* ls -a
* /bin/ls -al
2. For current directory:
* /bin/pwd
* pwd
3. For exit from shell:
* exit
* ctrl + d
* ctrl + c
